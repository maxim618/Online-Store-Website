# Online - Store - Website

Backend-приложение интернет-магазина, реализованное на Java и Spring Boot.

Проект создан как портфолио backend-разработчика и демонстрирует
практический опыт построения серверного приложения с фокусом на
архитектуру, безопасность, тестируемость и корректную бизнес-логику e-commerce.

---

## Технологический стэк

- Java 17
- Spring Boot 3
- Spring Security (JWT)
- Spring Data JPA (Hibernate)
- MySQL
- Redis-compatible storage (Valkey)
- Docker / Docker Compose
- JUnit 5
- GitHub Actions (CI)

---

## Функциональные возможности

- Аутентификация и авторизация пользователей
- Ролевая модель доступа (USER / ADMIN)
- REST API для управления товарами
- Корзина пользователя
- Оформление заказов
- Wishlist (избранные товары)
- Защищённая работа с пользовательскими данными
- Unit и integration тестирование
- Автоматическая проверка сборки в CI

---

## Особенности реализации

- Корзина реализована через абстракцию хранения и не привязана к конкретной технологии
- Поддерживаются разные реализации хранения корзины:
    - JPA (реляционная модель)
    - Redis-compatible storage (Valkey) для high-load сценариев - высокопроизводительной корзины
- Логика оформления заказов не зависит от способа хранения корзины
- Поведение системы подтверждено интеграционными тестами

---

## Платежи

Проект включает в себя специальный модуль "payments", разработанный для обеспечения масштабируемости в будущем
и возможного выделения в отдельный микросервис.

Ключевые характеристики:
- Логика платежей изолирована от бизнес-сервисов
- Внешние поставщики платежей абстрагируются через интерфейс `PaymentProvider`
- Для разработки и тестирования используется фиктивный платежный провайдер
- Данные карты не хранятся и не обрабатываются внутри приложения (совместимо с PCI DSS)
